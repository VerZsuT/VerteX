# Примеры кода

## Объявление функции

### Без аргументов

```
функция привет()
{
	печать("Привет");
}
```

### С аргументами

```
функция привет(имя)
{
	печать("Привет, " + имя);
}
```

## Импорт

### Ссылок

```
использовать ссылки "links.vxls";
```

### Стандартных типов

**links.vxls**:
```
Система = System
удалитьФайл = File.Delete
```

**main.vx**:
```
использовать Система;
удалитьФайл("Brain.exe")$
```

### Dll файлов

**links.vxls**:
```
Пространство = MyNamespace
мойМетод = MyMethod
```

**main.vx**:
```
импорт "myLibrary.dll";
использовать Пространство;

мойМетод("Привет");
```

## Ссылочный файл

**links.vxls**:
```
Имя = Name
Переменная = Variable
```
